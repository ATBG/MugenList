Mood: premium, cinematic, minimal chrome, fast-scanning. Provide top-down desktop frame (1440×900 area) with crisp UI elements and small layered depth (cards slightly elevated). Include annotations: sidebar width 240px expanded, poster card 190×285, progress bar height 3px, badge height 26px, countdown digits large and tabular.

Detailed guidance
- Frame: Desktop composition sized for 1440×900.
- Layout:
  - Left: collapsible sidebar (64px collapsed, 240px expanded) with vertical icon stack and subtle glass/blur.
  - Top: thin topbar (56px) with left-aligned search, center-left quick actions, right-aligned avatar and sync indicator.
  - Main area: hero + rails + grid stack centered; maintain comfortable side gutters.
- Cards:
  - Poster aspect ratio 2:3, poster card size 190×285 recommended for desktop grid.
  - Rounded corners 16–18px, subtle border (rgba white, 6%), soft shadow.
  - Status pill top-left (26px high), small dot + uppercase label.
  - New-ep badge top-right with optional tiny pulse for emphasis.
  - Footer: small progress bar (3px) with indigo → lavender gradient, left progress text (e.g., 12/24), right percent.

- Countdown overlay:
  - Monospace / tabular numbers, bold, large; label “NEXT EPISODE” above digits.
  - Display format: DD:HH:MM:SS (zero-padded)
  - Backdrop-blur small (<= 8–12px), translucent dark overlay to preserve poster visibility.
- Colors & Typography:
  - Background: #0B0E14
  - Card gradient: #161a2a → #0d0f18
  - Accent gradient: #5D5FEF → #A5A6F6; cyan highlight: #22D3EE
  - Typography: Inter for UI; Noto Sans JP for Japanese titles if present.
- Motion:
  - Hover transitions 180–320ms, cubic-bezier(0.16,1,0.3,1).
  - Poster zoom easing 600ms for a cinematic feel.
